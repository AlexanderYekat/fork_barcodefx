При формировании файла для ввода в оборот при трансграничной торговле обязательно указывать ТНВЭД код для каждого товара.

Программа берет данную информацию из каталога товара из столбца V_CLASS_TNVED.
Чтобы подключить каталог товаров вам необходимо сделать выгрузку товаров из ГС1 и указать его в Настройки -> Каталог товаров.

Так же берется информация о сертификате, его дате и типе из столбцов V_PROD_CERT_TYPE,V_PROD_CERT_NUMBER,V_PROD_CERT_ISSUE_DATE.
Согласно схеме эти данные не обязательны.

Цена и НДС тоже обязательны согласно схеме. По умолчанию заполняются нулями.

Чтобы брались из каталога товаров вам нужно в файле каталога товаров добавить столбцы COST и VAT.
Тогда программа будет подтягивать их оттуда.
